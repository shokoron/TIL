# 23が素数かどうかを判定するプログラム。
- ある数nより小さい全ての素数でnが割り切れない場合、nは素数であると言える。
- 23より小さい素数の配列を用意。
- 繰り返しの中で割り切れなかった回数をカウントしておき、「全ての素数で割り切れなかったかどうか」の判定で利用する。

```ruby
primes = [2,3,5,7,11,13,17]
count = 0

primes.each do |p|
   if 23 % p == 0
     count = count + 1
   end
end

if count == 0
  puts "カウントが0の場合、23は素数です"
else
  puts "23は素数では無いです"
end
